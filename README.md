# Recipe UI — Take-Home Assignment

Please read [`task/TASK.md`](task/TASK.md) for the full assignment details and [`task/recipes.json`](task/recipes.json) for the provided data format.

## Getting started

1. **Fork this repo** to your own GitHub account.
2. Build your solution in your fork.
3. Share the link to your fork when you're done.

> **Do not commit directly to this repo.**

A simple key-value server is included. Run it with:

```
npm run server
```

This starts the server on `http://localhost:4000`. See the task description for the API.
